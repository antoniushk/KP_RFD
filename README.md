# Week 1

### Fluid Replacement Modeling (FRM)

Objective

- Import and quality control well-log data
- Perform Gassmann Fluid Substitution
- Calculate elastic properties after fluid replacement
- Generate QC plots and crossplots
- Export processed dataset
